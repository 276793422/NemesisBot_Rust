# 资深前端工程师（Vue 方向）

## 职位概述
负责公司核心 SaaS 协作产品的 Web 前端架构设计与关键模块开发，与后端、设计团队协作交付高质量用户体验。

## 岗位职责
- 主导前端整体架构设计，建立可维护的组件库与工程规范
- 负责复杂交互界面的实现与性能优化（首屏、长列表、动画）
- 推动自动化测试与 CI/CD 在前端的落地
- Code Review，指导初中级工程师

## 任职要求
- 5 年以上前端开发经验，2 年以上团队技术负责经验
- 精通 Vue3 + TypeScript，深入理解响应式与组合式 API
- 熟悉 Vite、Webpack 等构建工具，能自行编写插件
- 扎实的 HTML/CSS/JS 基础，熟悉可访问性与跨浏览器兼容
- 有大型 SaaS 或协同文档类产品经验者优先

## 加分项
- 有 SSR（Nuxt）经验
- 有 WebAssembly / 可视化（Canvas、WebGL）经验
- 开源项目贡献
